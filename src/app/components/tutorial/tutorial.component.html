Searching promiscuous proteins in ProtMiscuity

There are many ways to find your protein in ProtMiscuity. The search is set by default to protein entries. The user should specify a UniProt ID or a common protein name in the the search field and click the search button. After the search, ProtMiscuity will show the results page with one card per protein. The user can press the desired card to retrieve all the information available for the protein.

Alternative search
  • By organism

    • The user can also search for all proteins related with the organism of interest by selecting 'Organism' from the drop-down menu

  • By reaction

    • The user can also search for all the proteins having this promiscuous or canonic reaction.

After a search, the results page show a card where you can press in order to get the all the information available of this protein.

Visualizing protein data

All the information is collapsed and listed in five items in which the user can search for the specific information of interest: general information, sequence and active sites information, structural information, including conformational diversity, and external links.

General Information

The general information section includes: the protein's common name, the UNIPROT and the KEGG IDs, the organism from which the protein comes from and the protein family name.
